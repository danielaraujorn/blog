SOBRE

  • O Blog de IoT é um site mantido por Daniel Araújo e Judson Costa. Todos com o objetivo principal de divulgar e ensinar como utilizar a plataforma SaIoT. Além disso, mostrar ferramentas, bibliotecas, APIs, tutoriais do mundo do software livre e de código aberto para as pessoas.

    O SaIoT (Smart Automation using Internet of Things) é uma plataforma genérica para internet das coisas que faz parte do projeto SmartMetropolis da UFRN.
